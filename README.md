# **Netflix Clone**

This is a Netflix clone built with React, designed to mimic the look and feel of Netflix, providing an interactive movie selection experience. The project utilizes Axios for fetching movie data from an API and allows users to browse movies in a smooth and user-friendly interface.

## **Features**

- Beautiful and responsive design  
- Interactive movie banners and sections  
- Movie selection with arrows for navigation  
- Fetching data dynamically with Axios from a movie API  
- Stylish header and footer with seamless navigation  

## **Tech Stack**

- **Frontend:** React, Axios, CSS  
- **State Management:** React Hooks  
- **Styling:** Custom CSS  
- **API Integration:** Movie API 

## **Setup Instructions**

```bash
# Clone the repository
git clone https://github.com/mauranyathi/netflix-clone.git

# Navigate to the project directory
cd netflix-clone

# Install dependencies
npm install

# Start the development server
npm start
